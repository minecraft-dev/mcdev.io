What is Minecraft Dev IntelliJ

All Major Java Minecraft Platforms

Experience first class support for all of the major Java Minecraft development platforms, including Bukkit and derivatives such as Spigot and Paper, Sponge, Forge, LiteLoader, and BungeeCord. All this with support for Java, Kotlin, and Scala.WIP

Dynamic Features

Whether you develop for Forge, Bukkit, or Sponge, Minecraft Dev has project creation, new inspections, code generation, and code modification features for all of them.
